# TCTP / 星際公民繁體中文(台灣本地化修正)翻譯計畫
此計劃旨在針對遊戲內容、設定選項等內容進行台灣本地化用語修正，並透過高標準的專業學術資料來源與對遊戲世界觀塑造進行深入的研究實現了頂尖水準的翻譯。

## **重點改進內容**
- 對遊戲內文本翻譯進行**台灣用語本地化**
